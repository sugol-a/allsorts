# Allsorts
*WIP*
Some operators for Blender to optimise my workflow.

## Installation
Just clone this into your `addons` directory:

```shell
$ git clone https://github.com/sugol-a/allsorts.git ~/.config/blender/3.4/scripts/addons
```

## Operators

### Quickmirror
Interactively mirror objects
